# 專案 [SoundBar](https://sound-bar.tk/)
### 復刻 Podcast 平台 [SoundOn](https://www.soundon.fm)

## 版本
- Ruby：2.7.2
- Rails：6.1.3.2

## 開啟專案起手式
- `bundle install`
- `yarn install`
- `rails db:migrate`
- `foreman s -f ppf`

## 環境變數設定
- 複製 `/config/application.sample.yml` 中的內容
- 手動新增 `application.yml` 
- 依需求放入 AWS S3 & Facebook & Google 對應之金鑰
