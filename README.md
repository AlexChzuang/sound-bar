## 開啟專案起手式
- `bundle install`
- `yarn install`
- `rails db:migrate`
- `foreman s -f ppf`



## 環境變數設定
- 修改 `/config/application.sample.yml`  並確認有放入 ==`.gitignore`== 中
- 加入 AWS & Facebook 金鑰
