# README
SoundBar - 復刻 [SoundOn](https://www.soundon.fm)

## 版本
- Ruby：2.7.2
- Rails：6.1.3.2

## 開啟專案起手式
- `bundle install`
- `yarn install`
- `rails db:migrate`
- `foreman s -f ppf`

## 環境變數設定
- 修改 `/config/application.sample.yml`  並確認有放入 `.gitignore` 中
- 加入 AWS & Facebook & Google 金鑰
